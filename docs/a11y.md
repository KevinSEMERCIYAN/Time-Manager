# Accessibilité (A11y)

Mesures appliquées :
- Labels visibles associés aux champs (lecture d’écran).
- Boutons accessibles clavier (focus natif).
- Contrastes texte/boutons vérifiés sur fond clair.

Mesure à poursuivre :
- Ajouter des attributs aria-live pour les messages d’erreur.
- Tester navigation clavier (tab order) sur formulaires longs.
